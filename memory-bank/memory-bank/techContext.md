## Technology Context (techContext.md)

This document outlines the technologies, tools, and processes used in the development of the Akıllı Ajanda Web Projesi.

### Technologies Used

*   **Backend Framework:** Laravel (MVC Architecture) - A PHP web framework for building robust and scalable web applications.
*   **Templating Engine:** Blade - Laravel's simple yet powerful templating engine.
*   **Frontend Framework:** Bootstrap - A popular CSS framework for responsive and mobile-first web development.
*   **Database:** PostgreSQL - An open-source, object-relational database system known for its reliability and data integrity.
*   **Frontend JavaScript:** Vanilla JS - Using core JavaScript principles without external libraries for specific interactive elements and enhancements.
*   **Architectural Pattern:** Repository Service Pattern - For abstracting data access logic and promoting code reusability and testability.
*   **Version Control:** Git - Distributed version control system for tracking code changes and collaboration.

### Software Development Tools

*   **Integrated Development Environment (IDE):** PHPStorm (or equivalent - VS Code, Sublime Text) - A dedicated PHP IDE with features like code completion, debugging, and refactoring.
*   **Database Management Tool:** Dbeaver (or equivalent - pgAdmin) - A universal database tool for managing PostgreSQL databases.
*   **Command-Line Interface (CLI):** Bash (or equivalent) - For executing commands and scripts.
*   **Package Manager (PHP):** Composer - For managing PHP dependencies.
*   **Package Manager (JavaScript):** npm (Node Package Manager) - For managing JavaScript dependencies if additional frontend packages are required beyond Bootstrap's provided functionality (although minimal use is planned).
*   **API Testing Tool:** Postman/Insomnia - For testing API endpoints.
